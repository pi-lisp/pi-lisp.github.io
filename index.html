uwulisp

A lightweight, experimental Lisp interpreter written in Rust — with paths, intervals, Π-types, Σ-types, and Glue types. All the cubical goodies, tied up with a pretty purple bandana.

View on GitHub Get started

Get started

Make sure you have Rust and Cargo installed. Clone the repository and run the test harness containing the sample expressions: 

cargo run

Also works as an interactive REPL, batch stdin pipe, or file runner — e.g. cargo run path/to/file.lisp

Why uwulisp

I

Paths & intervals

First-class cubical paths over I = [0, 1], with path, papply, and refl.

Π Σ

Π and Σ types

Dependent function and pair types, built right into the language as core forms.

⧉

Glue types

glue and unglue encode equivalences between types, cubical-style.

#

De Bruijn core

Surface syntax with named variables compiles to a clean De Bruijn–indexed core.

λ

Macros & lexical scoping

Standard Lisp essentials — defmacro, let, closures, and arithmetic.

{}

Gradual typing

Bidirectional typechecking with __Any__, __Type__, and friends.

A taste

; doubling glue equivalence
(define double (lambda (x) (* x 2)))
(define gv (glue 21 double))
(unglue gv)   ; => 42

; linear interpolation path
(define interp (path (i) (+ (* (- 1 i) 1) (* i 5))))
(papply interp 0.5)   ; => 3.0
