## 왜 uwulisp인가요?

<CardGrid>
  <Card title="큐비컬 타입 이론" icon="setting">
    표면 수준의 Lisp API를 통해 접근 가능한 내부 큐비컬 타입 이론(CTT) 커널을 제공합니다. 구간, 의존 함수(Π-타입), 의존 쌍(Σ-타입), 경로 타입, 합성, Glue 타입과 같은 고급 타입 구조를 포함합니다.
  </Card>
  <Card title="JIT 어셈블리 실행" icon="rocket">
    평가기에서 직접 인라인 x86-64 NASM 스타일 명령어 목록 또는 외부 어셈블리 파일을 어셈블하고 JIT 실행하여, RAX 레지스터에서 계산된 결과를 Lisp 숫자로 네이티브 반환합니다.
  </Card>
  <Card title="S-표현식 문법" icon="codePen">
    일반적인 기본 요소를 지원하는 표현식 트리를 구성하는 간단한 문법으로 구동됩니다. 인용, 정의, 조건부 평가, 텍스트 대체 매크로 등 전통적인 Lisp 구조를 포함합니다.
  </Card>
  <Card title="UwU 스타일" icon="heart">
    집약적인 수학적 논리에 귀엽고 표현력 있는 시각을 더합니다. 모든 큐비컬 기계론과 타입 검사 기능이 친근한 보라색 반다나를 두른 고양이 미학으로 깔끔하게 포장되어 있습니다.
  </Card>
</CardGrid>

## 시작하기

Rust와 Cargo가 설치되어 있는지 확인하세요. 저장소를 클론하고 샘플 표현식이 포함된 테스트 하네스를 실행하세요:

```bash
git clone https://github.com/uwulisp/uwulisp.git
cd uwulisp
cargo run
```
