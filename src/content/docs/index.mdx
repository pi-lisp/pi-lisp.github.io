## Why uwulisp?

<CardGrid>
  <Card title="Lisp at the surface" icon="seti:default">
    Write definitions, lambdas, conditionals, quotes, macros, lists, strings,
    numbers, and evaluator experiments using familiar parenthesized forms.
  </Card>
  <Card title="Cubical ideas inside" icon="puzzle">
    Work with intervals, path types, composition, dependent functions,
    dependent pairs, universes, variables, and Glue types.
  </Card>
  <Card title="Assembly when needed" icon="rocket">
    Use inline x86-64 NASM-style instruction lists or external assembly files
    and receive the value returned through RAX.
  </Card>
  <Card title="Two mascots, one language" icon="heart">
    Lispy the cat and Lispu the puppy give the project its softer edge:
    part guides, part guardians, fully committed to balanced parentheses.
  </Card>
</CardGrid>

<Aside type="tip" title="Run the interpreter">
  Clone the interpreter repository and start it with Cargo:

  ```bash
  git clone https://github.com/uwulisp/uwulisp.git
  cd uwulisp
  cargo run
  # if use nix
  nix shell github:username/uwulisp
  uwulisp
  ```
</Aside>
