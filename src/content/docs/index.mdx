## Why uwulisp?

<CardGrid>
  <Card title="Cubical Type Theory" icon="setting">
    Features an internal cubical type theory (CTT) kernel accessible via a surface-level Lisp API . Includes advanced type constructs like intervals, dependent functions (Π-types), dependent pairs (Σ-types), path types, composition, and Glue types.
  </Card>
  <Card title="JIT Assembly Execution" icon="rocket">
    Assembles and JIT-executes inline x86-64 NASM-style instruction lists or external assembly files directly from the evaluator, returning computed results from the RAX register natively as Lisp numbers.
  </Card>
  <Card title="S-Expression Syntax" icon="codePen">
    Powered by a simple grammar that builds an expression tree supporting common primitives . Includes traditional Lisp structures such as quotes, definitions, conditional evaluation, and textually-substituted macros .
  </Card>
  <Card title="UwU Style" icon="heart">
    Brings a cute, expressive perspective to intensive mathematical logic . All of the foundational cubical mechanics and type-checking capabilities come wrapped up neatly in a friendly, purple-bandana-wearing cat aesthetic .
  </Card>
</CardGrid>

## Get started

Make sure you have Rust and Cargo installed. Clone the repository and run the test harness containing the sample expressions:

```bash
git clone https://github.com/uwulisp/uwulisp.git
cd uwulisp
cargo run
```
